# 🌤️ Api Weather App - Frontend

Una aplicación moderna y limpia para consultar el clima en tiempo real, inspirada en la interfaz de Google Weather. Desarrollada con React, Vite y estilizada mediante CSS Modules para garantizar modularidad y buenas prácticas.

## ✨ Características
- Búsqueda de clima por ciudades en tiempo real.
- Diseño minimalista, adaptable y elegante utilizando CSS Modules.
- Integración estructurada con componentes independientes (`WeatherForm`, `WeatherCard`).
- Consumo de datos meteorológicos precisos.

## 🛠️ Tecnologías utilizadas
- **React 18** (Estructura y hooks)
- **Vite** (Entorno de desarrollo rápido)
- **CSS Modules** (Estilos encapsulados)
- **Lucide React** (Iconografía limpia)
